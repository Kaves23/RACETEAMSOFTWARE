📋1. Overview

Inventory Management gives you full visibility and control over all consumable stock, spare parts, tyres, tools, and supplies used by the race team. Unlike Assets (which tracks individual unique items), Inventory tracks items by quantity — how many you have, where they are stored, and when you need to reorder.

💡 Key Features:
  • Track stock quantities across multiple physical locations
  • Set reorder points and get low-stock alerts
  • Organise items by category, supplier, and location
  • Import bulk stock from CSV files
  • Sync stock levels directly from your Shopify store
  • Perform bulk operations on multiple items at once

🖥️2. Page Layout

The Inventory page is divided into three main panels:

Left Panel — Filters
  • Stock status quick-filters
  • Category list with item counts
  • Location list with item counts
  • Supplier list with item counts
Middle Panel — Items List
  • Search bar and sort controls
  • Table or card view toggle
  • Selectable rows for bulk actions
  • Colour-coded stock status
Right Panel — Item Details / Edit Form
  • Displays full details of the selected item
  • Shows location distribution (quantities per location)
  • Shows which boxes contain this item
  • Edit all fields and save directly from this panel

Top Action Buttons

  • 🛍️ Shopify Sync — Pull stock levels from your Shopify store
  • 📥 Import CSV — Bulk-import items from a spreadsheet
  • ➕ Add Item — Create a single new inventory item

🔍3. Browsing & Filtering Inventory

1Use the Quick-Filter Buttons

At the top of the left panel, four buttons give you instant filtered views:

  • All Items — Show everything
  • Low Stock — Items below their reorder point
  • Out of Stock — Items with zero quantity
  • Expiring Soon — Items with an upcoming expiration date

Each button shows a count badge so you always know how many items are in each state.

2Filter by Category, Location or Supplier

Click any category, location, or supplier in the left panel to filter the list to only those items. The item count next to each entry updates in real time as you apply filters.

3Search and Sort

Use the Search bar at the top of the middle panel to find items by name, SKU, or part number. Use the Sort dropdown to order results by Name, Quantity, Category, Location, or Last Used.

4Switch Between Table and Card View

Use the view-toggle icons to switch between Table view (rows of data, best for comparing many items) and Card view (compact tiles, good for a quick visual scan).

Understanding Stock Status Colours

● Adequate — Stock is above the reorder point. No action needed.
● Low Stock — Quantity is at or below the reorder point. Consider ordering more soon.
● Out of Stock — Quantity is zero. Reorder immediately.

➕4. Adding a New Item

1Click "Add Item"

Click the ➕ Add Item button in the top-right corner. The right panel will clear and present a blank form.

2Fill in Required Fields

At minimum you must fill in:

  • Item Name — A clear, searchable name (e.g. "M6 Hex Bolt 30mm", "Brake Fluid DOT 4 1L")
  • Category — Select from the dropdown (Consumables, Spares, Tyres, Tools, etc.)
3Set Location Distribution

In the Location Distribution section, click Add Location for each physical location where this item is stored. For each entry, choose the location from the dropdown and enter the quantity held there. You can add as many location entries as needed — the total quantity is calculated automatically.

💡 Tip: If the same item is split across Warehouse A, Warehouse B, and a race truck, add three separate location entries. The total quantity shown on the item card will be the sum of all locations.
4Fill in Optional Details

Complete as many of the following as possible for accurate tracking:

  • SKU / Part # — Manufacturer or internal part number
  • Unit of Measure — Default is "ea" (each); change to kg, L, m, etc. as appropriate
  • Min Qty (Reorder Point) — The quantity at which the system flags the item as Low Stock
  • Supplier — Who you purchase this from
  • Lead Time (days) — How many days it takes to receive a new order
  • Unit Cost — Purchase price per unit
  • Description / Notes — Any extra information
  • Auto-reorder — Tick this box to mark the item for automatic reordering workflows
5Save the Item

Click 💾 Save Item. A green success notification will confirm the item has been created and it will immediately appear in the list.

⚠️ Important: Item Name and Category are required. The form will not save without them.

✏️5. Editing an Existing Item

1Select the Item

Click on any item row or card in the middle panel. The right panel will load that item's current details.

2Make Your Changes

Edit any field directly in the right panel. Common changes include:

  • Adjusting quantities after stock take
  • Updating supplier or cost information
  • Adding or removing a location entry
  • Changing the reorder threshold
3Save Changes

Click 💾 Save Item. Changes are saved to the database immediately and the list refreshes.

⚠️ Remember: Changes are not saved until you click Save. Clicking elsewhere or selecting a different item will discard unsaved edits.
4Deleting an Item

With an item selected, click Delete Item at the bottom of the right panel. A confirmation prompt will appear before anything is permanently removed.

⚠️ Warning: Deletion is permanent. Consider setting quantity to zero instead if you may need the item record again in future.

📍6. Location Distribution

Race team inventory is rarely all in one place. The Location Distribution system lets you track exactly how much of each item is held at each physical location.

1How It Works

Each item can have multiple location entries. Each entry records a location (e.g. Warehouse, Race Truck 1, Workshop) and a quantity at that location. The total quantity shown on the item is the sum of all location entries.

2Adding a Location Entry

In the item edit form, scroll to the Location Distribution section and click Add Location. Select the location from the dropdown and enter the quantity. Repeat for each location that holds stock.

3Removing a Location Entry

Click the remove (×) button on the right side of any location entry row. The total quantity will recalculate automatically.

4Viewing Boxes Containing This Item

Below the Location Distribution section, the right panel also shows a Boxes Containing This Item list. This tells you which packed boxes include this part, how many units are in each box, and where that box is located — useful when packing for an event.

⚡7. Bulk Actions

When you need to update, move, or delete multiple items at once, bulk actions save significant time.

1Select Multiple Items

Tick the checkbox on the left of each item row you want to act on. Use the Select All checkbox at the top of the list to select every item currently visible (including any active filter).

2The Bulk Actions Toolbar Appears

As soon as one or more items are selected, a blue toolbar slides in at the top of the list showing the count of selected items and four action buttons:

  • Update Location — Move selected items to a new location
  • Update Supplier — Reassign selected items to a different supplier
  • Delete Selected — Permanently remove all selected items
  • Clear Selection — Deselect everything and dismiss the toolbar
3Confirm the Action

For destructive actions (Delete Selected) a confirmation dialog will appear. For move/update actions, a selection dialog will prompt you for the new value before applying to all selected items.

⚠️ Warning: Bulk delete cannot be undone. Double-check your selection count before confirming.

📥8. Importing from CSV

Use the CSV Import feature to add or update many items at once from a spreadsheet.

1Prepare Your Spreadsheet

Create a CSV file with column headers matching the inventory fields. Recommended columns:

  • name — Item name (required)
  • sku — SKU or part number
  • category — Category name
  • quantity — Total quantity on hand
  • unit — Unit of measure
  • min_qty — Reorder threshold
  • supplier — Supplier name
  • unit_cost — Cost per unit
  • description — Notes
2Click "Import CSV"

Click the 📥 Import CSV button and select your file. The system will parse the file and show a preview before importing.

3Review and Confirm

Check the preview for any errors or mismatched columns. Confirm to import. Items that already exist (matched by SKU) will be updated; new items will be created.

💡 Tip: Always start with a small test file of 5–10 items to verify that your column headers are mapped correctly before importing hundreds of rows.

🛍️9. Shopify Sync

If your team sells products through a Shopify store, you can pull live stock levels directly into the inventory system.

1Open the Shopify Sync Modal

Click 🛍️ Shopify Sync in the top navigation. The Shopify Sync modal will open.

2Enter Your Shopify Credentials

Fill in:

  • Shop Name — Your Shopify store subdomain (e.g. mystore from mystore.myshopify.com)
  • Admin API Access Token — Your shpat_ token from Shopify Admin → Apps → Private Apps

Click Test Connection to verify the credentials before syncing.

3Choose Sync Options
  • Update existing items — Match by SKU and update quantities on items already in the system
  • Sync location inventory levels — Import Shopify location-specific quantities
4Start the Sync

Click Start Sync. A progress bar will show the sync in real time. When complete, results will show the number of items synced and any errors encountered.

💡 Tip: Run a test connection first every time — Shopify tokens can expire or be revoked.

📖10. Field Reference Guide

Item Name * Required
A clear descriptive name. Include model numbers or key specs where helpful. Example: Brake Fluid DOT 4 1L, M8 Nyloc Nut (pack 50)
SKU / Part #
Manufacturer or internal part number. Used for Shopify matching and CSV imports.
Category * Required
Organises items in the left panel filter. Options include Consumables, Spares, Tyres, Tools, and any custom categories you have added.
Quantity
Auto-calculated from all Location Distribution entries. Cannot be edited directly — adjust individual location quantities instead.
Unit of Measure
How the item is counted. Default: ea (each). Other common values: kg, L, m, pair, set
Min Qty (Reorder Point)
When stock falls to or below this number the item is flagged as Low Stock and appears in the Low Stock filter.
Location Distribution
One or more location + quantity pairs. Each entry records stock held at a specific physical location.
Supplier
Who supplies this item. Used for the Supplier filter in the left panel.
Lead Time (days)
Days from order to delivery. Used for planning ahead of events.
Unit Cost
Purchase price per unit for budget tracking.
Description / Notes
Any additional information — specifications, storage notes, usage guidelines.
Auto-reorder
Tick to flag this item for automated reorder workflows. Does not place orders automatically; flags the item for review when below minimum quantity.
