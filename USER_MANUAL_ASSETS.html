📋 1. Overview

The Asset Management system allows you to track and organize all physical assets used by your race team. This includes tools, parts, equipment, and any other valuable items that need to be monitored.

💡 Key Features:
  • Track asset location, status, and value
  • Generate QR codes and barcodes for quick scanning
  • Monitor maintenance schedules
  • Link assets to boxes for event packing
  • View complete history of asset movements

➕ 3. Adding a New Asset

1 Click the "Add Asset" Button

In the top-right corner of the page, click the ➕ Add Asset button.

📸
Screenshot: Add Asset button location
Shows the button highlighted in the top-right toolbar
2 Asset Details Modal Opens

A large modal window will appear with multiple tabs for entering asset information:

  • 📋 Details - Basic information and technical specs
  • 📍 Location & Tracking - Current location and movement
  • 🔧 Maintenance - Service schedule and history
  • 📜 History - Movement and update logs
📸
Screenshot: Asset modal - Details tab
Shows the empty form with all fields visible
3 Fill in Basic Information

In the Basic Information section, enter the following:

🏷️ Barcode / ID: (Optional) Leave blank to auto-generate, or enter a custom barcode
📛 Name: * REQUIRED - Enter a descriptive name (e.g., "Front Wing Assembly", "Impact Drill 18V")
📦 Type: * REQUIRED - Select from predefined types (Equipment, Asset, Tool, etc.)
🏷️ Category: (Optional) Choose a category like Tools, Aerodynamics, Electronics, etc.
🚦 Status: Select current status: Available, In Use, Maintenance, or Retired
📄 Description: (Optional) Add any additional notes or specifications
💡 Tip: Use clear, searchable names. Include model numbers or key specifications in the name for easy identification later.

📖 5. Field Reference Guide

📝 Basic Information Fields

Barcode / ID
Type: Text (optional)
Description: Unique identifier for the asset. Leave blank for auto-generation.
Example: AST-2024-001, TOOL-WR-42
Name *
Type: Text (required)
Description: Descriptive name of the asset
Example: "Front Wing Assembly", "Cordless Impact Drill 18V"
Type *
Type: Dropdown (required)
Description: Classification of the asset
Options: Equipment, Asset, Tool, Part, Consumable
Category
Type: Dropdown (optional)
Description: Subcategory for organization
Options: Tools, Aerodynamics, Engine Parts, Wheels & Tyres, Electronics, Safety Equipment, etc.
Status
Type: Dropdown
Description: Current availability status
Options:
  • ✅ Available - Ready to use
  • ⚡ In Use - Currently being used
  • 🔧 Maintenance - Under repair or service
  • 📦 Retired - No longer in active service
Description / Notes
Type: Text area (optional)
Description: Additional details, specifications, or notes
Example: "Compatible with carbon fiber chassis only. Requires annual inspection."

🔬 Technical & Value Fields

Serial Number
Type: Text (optional)
Description: Manufacturer's serial number for warranty tracking
Example: SN-2024-ABC-12345
Weight (kg)
Type: Number (optional)
Description: Weight in kilograms for shipping and packing calculations
Example: 2.5, 15.75
Value (USD)
Type: Number (optional)
Description: Purchase price or replacement value for insurance
Example: 1500.00, 250.50

📍 Location & Tracking Fields

Location
Type: Dropdown
Description: Physical location of the asset
Options: Warehouse, Workshop, Truck, Track, Event, etc.
Box
Type: Dropdown
Description: If packed, which box contains this asset
Note: Links to Box Packing system
Movement Note
Type: Text
Description: Context for location changes
Example: "Moved to truck for Silverstone event"

🔧 Maintenance Fields

Last Maintenance Date
Type: Date
Description: When service was last performed
Next Maintenance Due
Type: Date
Description: Scheduled maintenance date
Note: System will alert when date approaches

💡 6. Tips & Best Practices

🎯 Naming Convention

Use consistent, descriptive names that include:

  • Item type or function (e.g., "Drill", "Wing", "Scanner")
  • Key specifications (e.g., "18V", "Front", "Wireless")
  • Model or version if applicable

Good examples:

  • ✅ "DeWalt Cordless Impact Drill 18V DCF887"
  • ✅ "Front Wing Assembly - Carbon Fiber 2024"
  • ✅ "Tire Pressure Gauge Digital 0-60 PSI"

Avoid:

  • ❌ "Tool 1", "Part", "Equipment" (too generic)
🏷️ Use Categories Effectively

Categories help you quickly filter and find assets. Group similar items together:

  • Tools: Hand tools, power tools, measurement devices
  • Aerodynamics: Wings, body panels, aero components
  • Electronics: Sensors, wiring, data systems
  • Safety Equipment: Fire extinguishers, helmets, protective gear
📦 Keep Locations Updated

The system is only useful if location data is current!

  • Update asset location when packing for events
  • Mark items as "In Use" when assigned to a driver or crew member
  • Return items to "Available" after use
  • Use the Box Packing integration to automatically track locations
🔧 Schedule Maintenance

Prevent equipment failures by tracking maintenance:

  • Enter manufacturer recommended service intervals
  • Set "Next Maintenance Due" dates for critical equipment
  • Update "Last Maintenance Date" after service
  • Use status "Maintenance" for items being serviced
💰 Track Value for Insurance

Keep accurate values for insurance claims:

  • Enter purchase price or current replacement value
  • Update values annually or when items depreciate significantly
  • Include high-value items like specialized tools, racing parts, electronics
  • Use the Export function to generate inventory reports for insurance
🔍 Use Search Effectively

Quick tips for finding assets:

  • Search works on name, barcode, serial number, and description
  • Combine search with filters for precise results
  • Use the table view for sorting by column (click column headers)
  • Use the card view for visual browsing with images
📱 QR Codes for Efficiency

Print QR code labels for quick asset identification:

  • Generate QR codes from the Location & Tracking tab
  • Print and attach labels to physical assets
  • Scan QR codes for instant asset lookup
  • Useful for event packing and inventory checks
⚠️ Important Reminders
  • Always click Save after making changes
  • Required fields (marked with *) must be completed
  • Changes sync to the database immediately - all team members will see updates
  • Use the History tab to see who made changes and when
  • Deleted assets cannot be recovered - mark as "Retired" instead
