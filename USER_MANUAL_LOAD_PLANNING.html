📋1. Overview

Load Planning is where you decide which boxes go into which truck or trailer for a race event. The system gives you both a 2D zone layout for a fast visual overview and a 3D interactive view for precise spatial planning. Real-time capacity tracking prevents overloading.

💡 Key Features:
  • Assign packed boxes to named zones inside each truck or trailer
  • Real-time weight and volume capacity tracking
  • Switch between a fast 2D zone overview and a detailed 3D spatial view
  • Search highlights specific boxes or items across both views
  • Auto-Optimize automatically arranges boxes for efficient loading
  • Export a full packing list as CSV or PDF
  • Print barcode labels for all loaded boxes
ℹ️ Before you start: Boxes must be created and packed in the Box Packing module first. They will appear in the Load Planner automatically once created.

⚙️3. Setting Up a Load Plan

1Select an Event

Use the Select Event dropdown at the top of the page to choose the race or event you are loading for. This links the load plan to the correct event record for reporting and history.

2Select a Truck or Trailer

Use the Select Truck/Trailer dropdown to choose the vehicle you are packing. The right panel will update to show that truck's zones and its weight and volume capacity limits.

💡 Tip: Trucks and trailers with their zone configurations are managed in the Vehicles section of the system. If a truck is missing, check there first.
3Add Boxes from the Left Panel

All boxes that have been packed in the Box Packing module appear in the left panel. Boxes already loaded into this truck appear faded with a "loaded" indicator. Begin placing boxes into the truck zones as described in the following sections.

📐4. Loading Boxes — 2D Layout View

The 2D view shows the truck as a grid of named zones (e.g. Front, Middle, Rear, Driver's Side, Passenger Side). This is the fastest view for assigning boxes to areas of the truck.

1Drag a Box into a Zone

Click and hold a box card in the left panel, then drag it over one of the zone areas in the right panel. The zone highlights in blue to indicate it is a valid drop target. Release to place the box.

2Move a Box Between Zones

You can also drag a placed box directly from one zone to another within the truck layout. Hover over the box until it shows a grab cursor, then drag to the new zone.

3Expand a Zone to See Contents

Click on any placed box within a zone to expand it and see all the items packed inside it. Click again to collapse.

4Remove a Box from the Truck

Click the remove button (🗑️) on a placed box card inside the zone. The box returns to the available list in the left panel.

Capacity Progress Bars

Each zone shows a capacity indicator bar beneath it:

  • Blue / Green — Capacity is fine
  • Orange / Yellow — Approaching limit, add carefully
  • Red — Over capacity, remove boxes from this zone

🧊5. Loading Boxes — 3D View

The 3D view renders the truck and all placed boxes in an interactive three-dimensional space. Use it for precise spatial planning — especially for large loads where stacking order matters.

1Switch to 3D View

Click the 3D View toggle button in the top-right of the truck panel. The canvas renders the truck with all currently placed boxes shown as coloured 3D blocks.

2Navigate the 3D View
  • 🖱️ Click and drag — Rotate the view around the truck
  • 🖱️ Scroll wheel — Zoom in and out
3Select and Move Boxes in 3D

Click on any box in the 3D view to select it (it will highlight). Then use the keyboard to move it:

  • ← → — Move box left or right
  • ↑ ↓ — Move box forward or backward
  • Page Up / Page Down — Move box up or down (stacking)
  • Delete — Remove the selected box from the truck
💡 Tip: Use Page Up / Page Down to accurately model multi-tier stacking inside the trailer. Heavier boxes should always be on the bottom tier.
4Switch Back to 2D

Click the 2D Layout toggle button at any time. All 3D placement changes are preserved when switching views.

🔍6. Searching the Load

The search bar in the left panel searches both box names and the items packed inside boxes. This is useful when you need to locate a specific part or piece of equipment across a large load.

1Type in the Search Bar

Enter any part of a box name, barcode, or item name. Results are highlighted immediately as you type.

2Colour-Coded Search Results

In both 2D and 3D views, the search applies visual highlighting:

  • Spring green highlight — This box or a box containing a matching item exactly matches your search
  • Grey / faded — Does not match; moved to background so matches stand out
  • Pulsing purple border on zone — The zone contains at least one matching box
3Clear the Search

Delete the text from the search bar or click the clear (×) button. All highlighting is removed and the full load reappears.

⚡7. Auto-Optimize

The Auto-Optimize feature automatically rearranges all currently loaded boxes across zones for better weight balance and space efficiency.

1Click "Auto Optimize"

Click the ⚡ Auto Optimize button in the top-right area of the truck panel. The system will analyse the current boxes and redistribute them across zones.

2Review the Result

After optimizing, review the new arrangement in the 2D or 3D view. The statistics cards will update to show the new weight distribution. Make any manual adjustments you need after the auto-place.

⚠️ Note: Auto-Optimize moves boxes between zones but does not guarantee the arrangement meets every physical constraint of your specific truck. Always do a final manual review before departure.

📋8. Exporting the Packing List

The packing list is a full manifest showing every box in the load and all items packed inside each box. Export it to share with the team or keep as an event record.

1Open the Export Modal

Click 📋 Export Packing List. A format selection dialog will open.

2Choose Your Format
  • 📊 CSV Spreadsheet — Downloads a .csv file. Open in Excel, Google Sheets, or Numbers for sorting and filtering. Useful for checking off items and doing stock takes.
  • 🖨️ PDF / Print — Opens the browser print dialog. Use to print a physical packing sheet to take trackside, or save as a PDF to share digitally.
💡 Tip: Print the packing list and keep a physical copy with the truck driver. Use it as an arrival checklist to confirm every box has arrived at the venue.

🖨️9. Printing Barcode Labels

1Click "Print Barcodes"

Click 🖨️ Print Barcodes in the top navigation. The browser print dialog opens with a formatted label sheet for all loaded boxes.

2Print or Save

Print directly to a label printer or standard A4 printer, or save to PDF. Attach labels to the outside of each box before loading so they can be scanned on arrival at the circuit.

💾10. Saving & Clearing Plans

Saving Click 💾 Save at any point to commit the current load plan to the database. The Load Status stat card will show Saved and a timestamp. Save frequently while building a large load.
Auto-save The system may auto-save after significant actions. The Load Status card will always show the time of the last save.
Clearing the Load Click 🗑️ Clear to remove all boxes from the truck and start over. A confirmation prompt will appear. This does not delete the boxes — it only removes them from this load plan. They return to the available list in the left panel.
Starting a New Event Load Select a different event from the Select Event dropdown. Each event has its own separate load plan, so switching events will show that event's saved plan without affecting others.
⚠️ Important: Always save before closing the browser tab. Any unsaved placement changes will be lost if you navigate away without saving.

💡11. Tips & Best Practices

Pack Boxes First Load Planning only works with boxes that have been set up and packed in the Box Packing module. Do all your packing there before coming to Load Planning.
Use 2D for Speed, 3D for Precision Start in 2D to quickly assign boxes to zones. Switch to 3D only when you need to plan stacking order or verify spatial fit.
Respect Weight Distribution Keep an eye on the Total Weight stat card and per-zone progress bars. Uneven loads can affect vehicle handling. Aim to keep heavy boxes low and centred.
Use Search to Find Critical Parts After completing a load, search for key items (e.g. "spare engine", "wet tyres") to confirm they are loaded and to quickly identify which box they are in.
Save Early and Often Building a full load plan for a race weekend takes time. Save after placing every few boxes so you never lose significant work.
Export Before You Leave Always export and save a copy of the packing list to PDF before the truck departs. This gives you an offline record if you lose internet access at the circuit.
Use Auto-Optimize as a Starting Point Run Auto-Optimize to get a sensible initial layout, then make manual adjustments for items that have specific placement requirements (fuel near exits, fragile items on top, etc.).
