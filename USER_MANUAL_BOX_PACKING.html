📋1. Overview

The Box Packing module is where you organise all your race equipment and assets into physical transport boxes before an event. You can build a complete inventory of what is packed in every box, making check-in and check-out at events fast and traceable.

💡 Key Features:
  • Drag-and-drop items from Assets or Inventory into any box
  • Track exactly what is in every box at all times
  • Create colour-coded Driver Boxes assigned to individual drivers
  • Bulk-move or empty multiple boxes in a single action
  • Full history log for every box — what was packed, when, and by whom
  • Print barcode labels for boxes

🖥️2. Page Layout

The page is split into three resizable panels. You can drag the dividers between panels to adjust their width.

Left — Items Available

All assets and inventory items available to pack. Switch between the Assets tab and the Inventory tab. Search and filter within each tab. Drag any item out of this panel and drop it into a box.

Middle — Boxes

All boxes in the system. Search, sort, and select boxes here. Click a box to see its contents in the right panel. Drag items onto a box card to pack them directly.

Right — Box Contents

Shows everything currently packed in the selected box. Drag items here from the left panel. Remove individual items with the trash button on each packed item.

Top Navigation Buttons

  • ➕ New Box — Create a new transport box
  • 🖨️ Print Label — Print barcode labels for boxes
  • 📜 Box History — View the full history of changes across all boxes
ℹ️ Data Freshness Indicator: The top bar shows a LIVE badge when data is up to date, or the time of last refresh. Click it to force-refresh all data from the server.

➕3. Creating a New Box

1Click "New Box"

Click the ➕ New Box button. The Create New Box modal will open.

2Fill in Box Details
Barcode / ID Auto-generated if left blank. Enter a custom value if you have pre-printed labels.
Box Name * Required A clear name such as Tools Box A, Spare Parts 1, or Aleks Race Gear.
Box Type Choose Regular Box for standard transport containers, or Driver Box for a box assigned to a specific driver (see Section 7).
Dimensions (L × W × H cm) Physical size of the box, used for volume calculations in the Load Planner.
Weight Capacity (kg) Maximum weight the box can hold. Used for capacity warnings.
Current Location * Required Where the box is physically located right now (e.g. Warehouse, Workshop, Race Truck 1).
Assign Driver Only appears when Box Type is set to Driver Box. Select the driver this box belongs to.
3Create the Box

Click Create Box. The new box appears immediately in the middle panel, ready to be packed.

📥4. Packing Items into a Box

There are two ways to pack items into a box.

1Method A: Drag and Drop
  1. In the left panel, find the item you want to pack. Use the search bar or switch between the Assets and Inventory tabs.
  2. Click and hold the item card — your cursor changes to a grab icon.
  3. Drag the item to the target box in the middle panel. The box card highlights when you hover over it.
  4. Release to drop the item into the box. The contents count badge on the box card updates immediately.
💡 Tip: You can also drag an item directly into the right panel (Box Contents area) while that box is selected, which shows the full contents list as a drop target.
2Method B: Select Box First, Then Add
  1. Click a box in the middle panel to select it. Its contents appear in the right panel.
  2. Find the item in the left panel.
  3. Drag the item into the right panel drop zone. The item appears in the packed contents list immediately.

Understanding Item Category Colours

Items in the left panel are colour-coded by category for quick identification:

Equipment Asset Consumable Filters Brakes Engine

Items that are out of stock appear faded and cannot be packed.

📤5. Removing Items from a Box

1Select the Box

Click the box in the middle panel. Its packed contents appear in the right panel.

2Remove an Individual Item

Find the item in the right panel contents list and click the red remove button (🗑️) on the right side of that item's row. The item is unpacked immediately and its quantity returns to the available pool in the left panel.

🧹6. Emptying a Box

Use the Empty Box action to unpack all items from a box in one go and move them to a specific location.

1Select the Box

Click the box you want to empty in the middle panel.

2Trigger the Empty Action

In the right panel, click the Empty Box button. The Empty Box — Select Unpack Location modal will open.

3Choose a Destination Location

Select the location where all unpacked items should be moved (e.g. Warehouse, Workshop). A preview list shows every item that will be affected.

Click Empty Box to confirm. All items are unpacked and their location is updated in one operation.

⚠️ This cannot be undone in a single click. Use the Box History log if you need to see what was in the box before emptying.

🚗7. Driver Boxes

Driver Boxes are special boxes assigned to an individual driver. They are visually distinct from regular boxes so you can instantly see which driver's gear is in which container.

Visual Distinction Driver Boxes appear with a glowing coloured border in the driver's assigned colour. A driver name badge is shown on the box card.
Creating a Driver Box When creating a new box, set the Box Type to Driver Box and select the driver from the Assign Driver dropdown.
Changing the Assigned Driver Click the driver badge displayed on the box card to reassign the box to a different driver.
Purpose Ideal for personal race gear, helmets, suits, and driver-specific spare parts that must travel with a specific driver.

⚡8. Bulk Box Actions

When preparing for or returning from an event, you often need to move or clear multiple boxes at once. Bulk actions handle this efficiently.

1Select Multiple Boxes

Tick the checkbox in the top-left corner of each box card in the middle panel. A blue bulk-actions toolbar will appear at the top of the middle panel showing the count of selected boxes.

2Choose an Action

Open the Bulk Actions dropdown in the toolbar and choose from:

  • 📦 → 📍 Move Boxes to Location — Change the physical location of all selected boxes (items stay packed inside)
  • 📦 → 📍 Unpack All to Location — Empty all selected boxes and move their contents to a chosen location
  • 🗑️ → 📍 Delete Boxes, Move Items to Location — Delete the box records, but reassign all their contents to a location first
  • 🗑️ Delete Boxes AND Items (Permanent) — Permanently deletes the boxes and all items inside them. Shown in red. Use with caution.
⚠️ Warning: The "Delete Boxes AND Items" action is irreversible. Only use it when you are certain those items should be permanently removed from the system.
3Confirm

A confirmation dialog will appear for any destructive action. A location picker will appear for actions that require a destination. Confirm to proceed.

📜9. Box History

Every significant change to a box is logged automatically — packing, unpacking, location changes, and bulk operations.

1Opening Box History

Click the 📜 Box History button in the top navigation. The Box History modal opens showing a chronological log of all box events.

2What the Log Shows
  • Timestamp of each action
  • Action description (e.g. "4 items added", "Moved to Warehouse", "Box emptied")
  • Location involved (where items came from or went to)
💡 Tip: Use history to reconstruct what was in a box before an accidental empty, or to confirm that a box was fully loaded before departure.

🖨️10. Printing Box Labels

1Click "Print Label"

Click the 🖨️ Print Label button in the top navigation.

2Print or Save

The browser's print dialog will open. You can print directly to a label printer, a standard printer, or save as a PDF for later use.

💡 Tip: Attach printed barcode labels to the outside of each box so they can be scanned at events for instant content verification.

💡11. Tips & Best Practices

Pack Before Every Event Always review and update box contents before departing. Use the box contents list as your pre-event checklist.
Use Descriptive Box Names Names like Tools Box A, Wet Weather Parts, and Engine Spares 1 are far easier to work with than Box 1, Box 2.
Always Set a Location Every box should have a current location set. This makes it easy to find where physical boxes are at any given time.
Use Driver Boxes for Personal Gear Helmets, suits, gloves, and driver-specific parts should live in a named Driver Box. This prevents mix-ups when multiple drivers share a truck.
Empty Boxes After Events Use the Empty Box action immediately on return to move everything back to the warehouse. This keeps your inventory location data accurate.
Resize Panels for Your Workflow Drag the dividers between panels to give more screen space to whichever panel you are currently working in.
