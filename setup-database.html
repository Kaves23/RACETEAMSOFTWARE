🗄️ PlanetScale Database Setup

1 Get Your Connection Details from PlanetScale

Go to https://app.planetscale.com and log in to your account.

  1. Click on your database: "raceteam-logistics-v5" (or whatever you named it)
  2. Click the "Connect" button in the top right corner
  3. From the dropdown, select "Connect with: Node.js"
  4. You'll see connection details like this:
const connection = mysql.createConnection({ host: 'aws.connect.psdb.cloud', username: 'xxxxxxxxxxxxxxxxx', password: 'pscale_pw_xxxxxxxxxxxxxxxxxxx', database: 'raceteam-logistics-v5' })
⚠️ Important: If you don't see a password, click "New password" to create one. Copy it immediately - you can't see it again!

        3
        Install Required Packages
      

      
Open your terminal and run these commands:

      

cd "/Users/John/Dropbox/RACE TEAM SOFTWARE V5"
npm init -y
npm install @planetscale/database dotenv express cors body-parser
npm install --save-dev nodemon
      

      
    


    

      

        4
        Test Database Connection
      

      
After creating your .env file and installing packages, test the connection:

      

node server/test-connection.js
      

      
You should see:

      

        ✅ Database connection successful!

        ✅ Query successful!

        Database: raceteam-logistics-v5
